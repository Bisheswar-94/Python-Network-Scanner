# Python Network Scanner

Python Network Scanner is a cybersecurity and network discovery tool built with Python. It scans a local network, detects active hosts, resolves hostnames, and performs fast multi-threaded scanning for reconnaissance and learning purposes.

---

## Features

- Scan local networks
- Detect active hosts
- Get IP addresses
- Resolve hostnames
- Multi-threaded fast scanning
- Clean terminal output
- Simple and beginner friendly

---

## Technologies Used

- Python 3
- Socket Programming
- Threading
- Queue
- Colorama
